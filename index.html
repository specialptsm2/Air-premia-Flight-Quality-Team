Workload (작업 부하)
구 분정 의판단 기준예 시비 고
None작업 요구 거의 없음단일 단순 task, 시간적 여유 충분, 인지 부하 거의 없음대기 상태, monitoring only부하 없음
Low단일 또는 단순 작업 수행시간 압박 거의 없음, 여유 있는 상황 인지 가능단일 또는 단순 작업 수행, 시간 압박 거의 없음, 여유 있는 상황 인지 가능, routine task, interruption 없음정상 여유
Medium여러 작업 동시 수행일정 수준의 time pressure, task prioritization 필요, 집중 필요한 상황multitasking, coordination 필요주의 필요
High작업 요구가 개인 능력에 근접하거나 초과강한 time pressure, 빈번한 interruption, situational awareness 유지 어려움task overload, emergency 대응과부하
Attention (주의 집중 상태)
구 분정 의판단 기준예 시비 고
Normal Scan체계적으로 여러 정보원 스캔계기와 외부 환경을 체계적으로 스캔하며 여러 정보원에 주의를 분산하여 상황인식을 유지하는 상태PFD / ND / engine / outside view를 주기적으로 확인, ATC / PM cross-check 정상, abnormal fixation 없음abnormal fixation 없음
Channelized특정 작업에 주의 집중특정 작업이나 정보원에 주의가 집중되어 다른 관련 정보에 대한 모니터링이 감소한 상태FMS programming에 집중하면서 altitude deviation monitoring 감소, abnormal checklist 수행 중 flight path monitoring 감소주의 협착
Fixation한 가지 요소에 고착됨단일 문제나 작업에 주의가 지속적으로 집중되어 변화하는 상황이나 다른 중요한 정보가 적절히 인지되지 않는 상태FMS 문제 해결에 집착, landing gear 문제 해결 중 altitude deviation 인지 못함위험 상태
Time Pressure (시간 압박)
구 분판단 기준상세 설명
None시간 여유 충분긴급한 결정이나 조치가 필요 없는 안정적인 상태
Low약간의 시간 제약일반적인 절차 수행에는 문제가 없으나 시간 관리가 시작되는 단계
Medium작업 우선순위 조정 필요가용한 시간 내에 작업을 마치기 위해 효율적인 우선순위 배분이 필요한 상태
High즉각적인 대응 요구매우 촉박한 시간 내에 결정과 행동이 이루어져야 하는 높은 압박 상태
Fatigue (피로도)
구 분정 의예 시
None피로 징후가 없으며 주의력과 인지 기능이 정상적으로 유지되는 상태alertness 정상, 집중력 저하 없음, 반응속도 정상
Low경미한 피로가 있으나 작업 수행과 상황 인식에는 거의 영향을 주지 않는 상태약간의 피로감, 집중력 유지 가능, 수행 능력 영향 미미, 장시간 근무 초기 단계
Medium피로로 인해 집중력과 주의 분배 능력이 감소하며 작업 수행에 일부 영향을 줄 수 있는 상태집중력 감소, 주의 분산 어려움, 반응속도 약간 저하, 장시간 비행 후반, 수면 부족
High심한 피로로 인해 주의력, 판단력 또는 상황인식이 현저히 저하된 상태alertness 크게 감소, monitoring 능력 저하, error risk 증가, severe sleep loss, 장시간 duty 후반
